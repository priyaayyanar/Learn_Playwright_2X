# VS Code Keyboard Shortcuts - Windows

| Category | Shortcut | Action |
|---|---|---|
| **General** | `Ctrl + Shift + P` | Command Palette |
| | `Ctrl + P` | Quick Open (Go to File) |
| | `Ctrl + Shift + N` | New Window |
| | `Ctrl + W` | Close Window/Tab |
| | `Ctrl + ,` | Open Settings |
| | `Ctrl + K Ctrl + S` | Keyboard Shortcuts |
| **File** | `Ctrl + N` | New File |
| | `Ctrl + O` | Open File |
| | `Ctrl + S` | Save |
| | `Ctrl + Shift + S` | Save As |
| | `Ctrl + K S` | Save All |
| **Editing** | `Ctrl + X` | Cut line (empty selection) |
| | `Ctrl + C` | Copy line (empty selection) |
| | `Ctrl + V` | Paste |
| | `Ctrl + Shift + K` | Delete Line |
| | `Ctrl + Enter` | Insert Line Below |
| | `Ctrl + Shift + Enter` | Insert Line Above |
| | `Alt + Up/Down` | Move Line Up/Down |
| | `Shift + Alt + Up/Down` | Copy Line Up/Down |
| | `Ctrl + /` | Toggle Line Comment |
| | `Shift + Alt + A` | Toggle Block Comment |
| | `Ctrl + D` | Add Selection to Next Find Match |
| | `Ctrl + Shift + L` | Select All Occurrences |
| | `Alt + Click` | Insert Cursor |
| | `Ctrl + Alt + Up/Down` | Add Cursor Above/Below |
| | `Ctrl + U` | Undo Last Cursor Operation |
| **Search** | `Ctrl + F` | Find |
| | `Ctrl + H` | Replace |
| | `F3` / `Shift + F3` | Find Next / Previous |
| | `Ctrl + Shift + F` | Search in Files |
| | `Ctrl + Shift + H` | Replace in Files |
| **Navigation** | `Ctrl + G` | Go to Line |
| | `Ctrl + Shift + O` | Go to Symbol in File |
| | `Ctrl + T` | Go to Symbol in Workspace |
| | `F12` | Go to Definition |
| | `Alt + F12` | Peek Definition |
| | `Ctrl + Shift + F10` | Peek References |
| | `Ctrl + -` / `Ctrl + Shift + -` | Go Back / Forward |
| **Editor** | `Ctrl + \` | Split Editor |
| | `Ctrl + 1/2/3` | Focus Editor Group 1/2/3 |
| | `Ctrl + K Ctrl + W` | Close All Editors |
| | `Ctrl + K Z` | Zen Mode |
| **Terminal** | `` Ctrl + ` `` | Toggle Integrated Terminal |
| | `` Ctrl + Shift + ` `` | New Terminal |
| | `Ctrl + Shift + C` | Copy Terminal Selection |
| | `Ctrl + Shift + V` | Paste in Terminal |
| **Debug** | `F5` | Start Debugging |
| | `Shift + F5` | Stop Debugging |
| | `F9` | Toggle Breakpoint |
| | `F10` | Step Over |
| | `F11` | Step Into |
| | `Shift + F11` | Step Out |
| | `Ctrl + Shift + D` | Show Debug Panel |
