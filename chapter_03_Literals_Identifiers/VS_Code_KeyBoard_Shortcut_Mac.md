# VS Code Keyboard Shortcuts - Mac

| Category | Shortcut | Action |
|---|---|---|
| **General** | `Cmd + Shift + P` | Command Palette |
| | `Cmd + P` | Quick Open (Go to File) |
| | `Cmd + Shift + N` | New Window |
| | `Cmd + W` | Close Window/Tab |
| | `Cmd + ,` | Open Settings |
| | `Cmd + K Cmd + S` | Keyboard Shortcuts |
| **File** | `Cmd + N` | New File |
| | `Cmd + O` | Open File |
| | `Cmd + S` | Save |
| | `Cmd + Shift + S` | Save As |
| | `Cmd + K S` | Save All |
| **Editing** | `Cmd + X` | Cut line (empty selection) |
| | `Cmd + C` | Copy line (empty selection) |
| | `Cmd + V` | Paste |
| | `Cmd + Shift + K` | Delete Line |
| | `Cmd + Enter` | Insert Line Below |
| | `Cmd + Shift + Enter` | Insert Line Above |
| | `Option + Up/Down` | Move Line Up/Down |
| | `Shift + Option + Up/Down` | Copy Line Up/Down |
| | `Cmd + /` | Toggle Line Comment |
| | `Shift + Option + A` | Toggle Block Comment |
| | `Cmd + D` | Add Selection to Next Find Match |
| | `Cmd + Shift + L` | Select All Occurrences |
| | `Option + Click` | Insert Cursor |
| | `Cmd + Option + Up/Down` | Add Cursor Above/Below |
| | `Cmd + U` | Undo Last Cursor Operation |
| **Search** | `Cmd + F` | Find |
| | `Cmd + H` | Replace |
| | `Cmd + G` / `Cmd + Shift + G` | Find Next / Previous |
| | `Cmd + Shift + F` | Search in Files |
| | `Cmd + Shift + H` | Replace in Files |
| **Navigation** | `Ctrl + G` | Go to Line |
| | `Cmd + Shift + O` | Go to Symbol in File |
| | `Cmd + T` | Go to Symbol in Workspace |
| | `F12` | Go to Definition |
| | `Option + F12` | Peek Definition |
| | `Shift + F12` | Peek References |
| | `Ctrl + -` / `Ctrl + Shift + -` | Go Back / Forward |
| **Editor** | `Cmd + \` | Split Editor |
| | `Cmd + 1/2/3` | Focus Editor Group 1/2/3 |
| | `Cmd + K Cmd + W` | Close All Editors |
| | `Cmd + K Z` | Zen Mode |
| **Terminal** | `` Ctrl + ` `` | Toggle Integrated Terminal |
| | `` Ctrl + Shift + ` `` | New Terminal |
| | `Cmd + C` | Copy Terminal Selection |
| | `Cmd + V` | Paste in Terminal |
| **Debug** | `F5` | Start Debugging |
| | `Shift + F5` | Stop Debugging |
| | `F9` | Toggle Breakpoint |
| | `F10` | Step Over |
| | `F11` | Step Into |
| | `Shift + F11` | Step Out |
| | `Cmd + Shift + D` | Show Debug Panel |
