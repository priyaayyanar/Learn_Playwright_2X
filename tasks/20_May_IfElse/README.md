# Task: If / Else Practice (20 May)

This folder contains hands-on practice tasks to strengthen understanding of JavaScript `if / else if / else` conditional statements.

---

## File Index

| # | File | Task | Description |
|---|------|------|-------------|
| 1 | `56_odd_or_even.js` | Odd or Even | Check if a given number is odd or even using the modulus operator |
| 2 | `57_student_grade_calc.js` | Student Grade Calculator | Assign letter grades (A, B, C, D, Fail) based on marks |
| 3 | `58_leap_year.js` | Leap Year Check | Determine if a given year is a leap year |

---

## Topics Practiced

- Using the modulus operator (`%`) inside conditions
- Multi-branch logic with `if / else if / else`
- Combining logical operators (`&&`, `||`) in conditions
- Real-world conditional problem solving

---

## Running the Tasks

```bash
node 56_odd_or_even.js
node 57_student_grade_calc.js
node 58_leap_year.js
```
